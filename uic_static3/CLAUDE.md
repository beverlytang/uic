# Medical website redesign

Using the content from uicmedcenter.com, design and build a medical, professional-looking redesigned site. Be sure to use as much of the content as possible.

## Visual Design

- Use the uic_logo.gif file in /images as the logo
- Also use the clinic.jpg file in /images in the Location section
- Use the jpg files in the /images folder where appropriate
- Responsive and mobile-first
- Professional and medically reputable looking
- Use expandable sections to include more details, as necessary
- Top navigation smoothly scrolls user down to the section
- Apply subtle tasteful animations on scroll, not too flashy

## Color Scheme

- Accent Red: `#B8292E` (submit buttons)
- Purple Primary: `#2F2F69` (headings)
- Light Blue Secondary: `#336699` (secondary buttons)
- Black Text: `#000000`

## Technical Notes

- Provide a live google map and a link to Directions of the location
- Use design tokens
- Make sure it complies with accessibility standards, including color contrast
- Provide a contact form
