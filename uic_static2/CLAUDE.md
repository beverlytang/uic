# Medical website redesign

Using the content from uicmedcenter.com, design and build a medical, professional-looking redesigned site. Be sure to use as much of the content as possible. Use the uic_logo.gif file as the logo. Also use the clinic.jpg file in the Location section.

## Color Scheme

- Accent Red: `#B8292E` (submit buttons)
- Purple Primary: `#2F2F69` (headings)
- Light Blue Secondary: `#336699` (secondary buttons)
- Black Text: `#000000`

## Technical Notes

- Provide a google map and a link to Directions of the location
- Ideally, keep all styles & javascript in index.html so it can be opened locally
